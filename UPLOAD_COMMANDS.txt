# Commands to Upload to GitHub

# 1. First, create the repository on GitHub.com with name: vidyo-password-manager

# 2. Then run these commands (replace YOUR_USERNAME with your actual GitHub username):

cd /home/alumbrados/Downloads/qudo/vidyo-password-manager

git remote add origin https://github.com/YOUR_USERNAME/vidyo-password-manager.git

git branch -M main

git push -u origin main

# That's it! Your repository will be uploaded to GitHub.
